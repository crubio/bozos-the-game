# untitled 2d game in gdscript
A tutorial game project built with Godot Engine.

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/crubio/bozos-the-game.git
   ```
2. Open the project in Godot.

## Features
- Player movement
- Enemy mobs
- Score and HUD
- Game over and restart

## Requirements
- Godot Engine 4.x
